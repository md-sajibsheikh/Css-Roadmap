১. CSS সিনট্যাক্স ও গঠন

CSS (Cascading Style Sheets) হল HTML উপাদানগুলোকে সাজানোর জন্য ব্যবহৃত। এর মৌলিক গঠন হল সিলেক্টর এবং ডিক্লারেশন ব্লক: 

        সিলেক্টর {
          প্রপার্টি: মান;
        }

উদাহরণ:

        h1 {
          color: blue;  /* রঙ নীল */
          font-size: 24px;  /* আকার ২৪ পিক্সেল */
        }

এখানে:

  • সিলেক্টর: h1 (সব <h1> উপাদানকে লক্ষ্য করে)
  • প্রপার্টিস: color ও font-size
  • মান: blue এবং 24px

২. CSS-এর প্রকার

  • ইনলাইন CSS: HTML উপাদানের মধ্যে সরাসরি style অ্যাট্রিবিউট দিয়ে যুক্ত করা হয়। 
                    <h1 style="color: blue;">হ্যালো বিশ্ব</h1>
  • অভ্যন্তরীণ CSS: HTML ডকুমেন্টের <head> অংশে <style> ট্যাগের মধ্যে লেখা হয়। 
                    <style>
                  h1 {
                    color: blue;
                  }
                </style>
  • এক্সটার্নাল CSS: একটি পৃথক .css ফাইলে লেখা হয় এবং HTML ডকুমেন্টে লিঙ্ক করা হয়। 
                    <link rel="stylesheet" href="styles.css">

৩. সিলেক্টর ও স্পেসিফিসিটি

সিলেক্টর হল প্যাটার্ন যা নির্দিষ্ট উপাদানগুলোকে লক্ষ্য করতে ব্যবহৃত হয়। কিছু সাধারণ সিলেক্টর: 

        p {
          color: green; /* সব প্যারাগ্রাফকে সবুজ রঙে সাজাবে */
        }
    
        .example {
          font-weight: bold; /* বোল্ড লেখা */
        }
    
        #unique {
          font-size: 30px; /* ৩০ পিক্সেল আকার */
        }

স্পেসিফিসিটি নির্ধারণ করে কোন CSS নিয়মটি কার্যকর হবে যখন একই উপাদানের জন্য বিভিন্ন স্টাইল প্রয়োগ করা হয়। স্পেসিফিসিটির ক্রম:

  • ইনলাইন স্টাইল (সর্বোচ্চ)
  • আইডি
  • ক্লাস, অ্যাট্রিবিউট, ও pseudo-classes
  • এলিমেন্ট ও pseudo-elements (সর্বনিম্ন)
        #header {
          color: red; /* আইডি সিলেক্টর */
        }
        .header {
          color: blue; /* ক্লাস সিলেক্টর */
        }
        h1 {
          color: green; /* এলিমেন্ট সিলেক্টর */
        }

যদি একটি <h1> আইডি header এবং ক্লাস header উভয়ই থাকে, তবে সেটি লাল (red) হবে কারণ আইডির স্পেসিফিসিটি বেশি।

৪. বক্স মডেল

বক্স মডেল হল CSS-এ উপাদানগুলো কিভাবে গঠিত হয় তা বর্ণনা করে। প্রতিটি উপাদান একটি বক্স হিসেবে বিবেচিত হয়, যা চারটি অংশ নিয়ে গঠিত:

  • কনটেন্ট: যেখানে টেক্সট এবং ইমেজ থাকে।
  • প্যাডিং: কনটেন্ট ও বর্ডারের মধ্যে ফাঁকা জায়গা (বক্সের ভিতরে)।
  • বর্ডার: প্যাডারের চারপাশের রেখা (যদি থাকে)।
  • মার্জিন: বর্ডারের বাইরের ফাঁকা জায়গা, অন্য উপাদানগুলো থেকে আলাদা করে।
        div {
          margin: 20px;  /* বাইরের ফাঁকা জায়গা */
          border: 2px solid black; /* বর্ডার */
          padding: 15px; /* ভিতরের ফাঁকা জায়গা */
        }

সারাংশ

  • CSS সিনট্যাক্স: সিলেক্টর + ডিক্লারেশন ব্লক
  • CSS প্রকার: ইনলাইন, অভ্যন্তরীণ, এক্সটার্নাল
  • সিলেক্টর: এলিমেন্ট, ক্লাস, আইডি
  • স্পেসিফিসিটি: কোন স্টাইল প্রয়োগ হবে তা নির্ধারণ করে
  • বক্স মডেল: কনটেন্ট, প্যাডিং, বর্ডার, মার্জিন
