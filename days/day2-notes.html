১. ফন্ট প্রপার্টিজ

CSS-এ টেক্সটের ফন্টের বৈশিষ্ট্য নির্ধারণ করতে কিছু মূল প্রপার্টি ব্যবহার করা হয়: 

        font-family: টেক্সটের জন্য কোন ফন্ট ব্যবহার করতে হবে তা নির্ধারণ করে।
        
        p {
          font-family: Arial, sans-serif; /* Arial ফন্ট, না থাকলে sans-serif */
        }
        
        font-size: টেক্সটের আকার নির্ধারণ করে।
        
        h2 {
          font-size: 32px; /* ৩২ পিক্সেল আকার */
        }
        
        font-weight: টেক্সটের ওজন বা ঘনত্ব নির্ধারণ করে (bold, normal, 100-900)।
        
        strong {
          font-weight: bold; /* বোল্ড লেখা */
        }
        
        font-style: টেক্সটের স্টাইল নির্ধারণ করে (normal, italic, oblique)।
        
        em {
          font-style: italic; /* ইটালিক লেখা */
        }

২. টেক্সট এলাইনমেন্ট ও ডেকোরেশন

টেক্সট কিভাবে সজ্জিত হবে তা নির্ধারণ করে:

        text-align: টেক্সটের অবস্থান (left, right, center, justify)।

        h1 {
          text-align: center; /* কেন্দ্রের দিকে সজ্জিত */
        }
        
        text-decoration: টেক্সটের উপরিভাগের সাজসজ্জা (underline, overline, line-through, none)।

        a {
          text-decoration: underline; /* আন্ডারলাইন */
        }

৩. ওয়েব ফন্টস

CSS-এ বিভিন্ন ওয়েব ফন্ট ব্যবহার করা যায়। Google Fonts এর মত সেবা ব্যবহার করে সহজেই ফন্ট যোগ করা যায়। 

        <link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">

        body {
          font-family: 'Roboto', sans-serif; /* Roboto ফন্ট ব্যবহার */
        }

৪. রেসপনসিভ টাইপোগ্রাফি

মোবাইল ও ডেস্কটপে ভিন্নভাবে দেখানোর জন্য, মিডিয়া কুয়েরি ব্যবহার করা হয়: 

        h1 {
          font-size: 24px; /* ছোট স্ক্রীনে */
        }

        @media (min-width: 768px) {
          h1 {
            font-size: 36px; /* বড় স্ক্রীনে */
          }
        }

সারাংশ

  • ফন্ট প্রপার্টিজ: font-family, font-size, font-weight, font-style
  • টেক্সট এলাইনমেন্ট ও ডেকোরেশন: text-align, text-decoration
  • ওয়েব ফন্টস: Google Fonts বা অন্যান্য উৎস থেকে ফন্ট ব্যবহার
  • রেসপনসিভ টাইপোগ্রাফি: মিডিয়া কুয়েরি ব্যবহার করে ভিন্ন স্ক্রীনের জন্য ভিন্ন ফন্ট সাইজ নির্ধারণ
